# Calculator 2026 Pro

Aplikasi kalkulator premium multi-platform (Flutter) — scientific, konverter, alat praktis, splash 3D, dan UI glassmorphism.

## Dokumentasi Proyek

| File | Format | Deskripsi |
|------|--------|-----------|
| [BLUEPRINT.md](BLUEPRINT.md) | Markdown | Blueprint lengkap (arsitektur, fitur, desain, roadmap) |
| [BLUEPRINT.docx](BLUEPRINT.docx) | Microsoft Word | Versi Word dari blueprint (buka di Word / LibreOffice) |

Regenerasi file Word setelah mengubah `BLUEPRINT.md`:

```bash
pip install python-docx
python scripts/generate_blueprint_docx.py
```

## Menjalankan Aplikasi

```bash
flutter pub get
flutter run
```

## Tech Stack

- Flutter / Dart 3.11+
- google_fonts, shared_preferences
